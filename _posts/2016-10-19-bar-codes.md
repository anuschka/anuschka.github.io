I reviewed the barcodes for various input material for the application and found out:

* Some input material has the QR
* Almost all input material has the barcode
* Very few reagents have no code (only manual Lot number and expiry date)

The application should be able to read barcodes and manual entry should be possible too.

I still need to find out how the barcode numbers are structured.
