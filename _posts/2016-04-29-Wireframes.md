Maksim and I decided it was time to create wireframes for the application. I started the exercise with the login and signup page that I already programmed in Django and then extended with the rest of the pages that I would need for the application consulting the data model that I already had setup in Django.

I wanted to use the <a href="https://moqups.com/"> Mocqup</a> software first but found it not too friendly. Maksim suggested <a href="https://balsamiq.com/"> Balsamiq </a> and so I ended up using it and found it really easy to use and friendly.

I made the decision to enter Cells from the CellPanel page since Cells can only belong to CellPanels. There is a separate page for Reagents and IDCards. These are also included in the menu buttons that appear on each page.

The main portal page is the Essey page where all the Esseys are displayed and new Esseys are added.
