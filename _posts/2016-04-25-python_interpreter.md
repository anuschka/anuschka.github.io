It can be handy to use Python's interactive interpreter to experiment with your Django application code or aspects of the framework itself.
The builin interpreter is just fine <code>manage.py shell</code> but iPython is even better.

To install it I ran <code>pip install ipython</code> and to run it <code>python manage.py shell -i ipython </code>.

iPython provide numerous features such as command-line numbering, automatic indentation, command history etc.
