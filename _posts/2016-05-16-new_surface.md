I got myself a new development machine <a href="https://www.microsoft.com/surface/en-us/devices/surface-pro-4">Surface Pro 4</a>. It was really time to do that. My old Dell laptop was getting rather slow and it was heavy to drag back and forth on a bike...

I really like my new Surface Pro 4! It is a rocket in my pocket <i class="fa fa-rocket" aria-hidden="true"></i>


I decided to install WIN10 as a host system with Ubuntu on a VMWare Player. My development environment is on Ubuntu 16.04LTS.

It took me a day or two to setup the how WIN10 machine with Balsamiq, McAfee AV, MS Office 2016, Skype, VMWare Player, WinSCP, Putty...

Then i had to Setup the Django environment on the new machine and clone my repositories from Github and install Atom with the linter and flake8 packages: `sudo apt-get install python3-flake8`. I also installed Zeal which is great for documentation.
